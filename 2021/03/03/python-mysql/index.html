Python管理Mysql数据库

  • python_mysql

使用 mysql-connector 来连接使用 MySQL数据库， mysql-connector 是 MySQL 官方提供的驱动器。需阅读者掌握一定的Mysql语法。

本章所讲

Python连接Mysql的语法和函数方法，其他的都是Mysql语句温习。

安装mysql.connector

pip install mysql-connector

demo.py

创建数据库连接

mysql.connector.connect() 返回一个数据资源句柄

创建数据库 CREATE DATABASE

mycursor = mydb.cursor：返回一个用来操控数据库的对象，也叫数据库游标
mycursor.execute(sql)：通过返回的数据库游标来执行sql语句


查看所有数据库 SHOW DATABASES

创建数据表 CREATE TABLE

因为要创建数据表，所以在连接数据库的时候要指定连接的哪个数据库 database = x
SQL语句：CREATE TABLE name (id CAHR(20))

查看所有表 SHOW TABLES

主键设置 ALTER TABLE

一般字段内作为唯一标识符的id值，不能出现重复的值，便于管理，也要依次递增，所以要把id设为主键
语法：
id INT AUTO_INCREMENT PRIMARY KEY
刚刚已经创建了数据表site，所以只需要在表中加入id主键即可
语句：
ALTER TABLE sites ADD COLUMN id INT AUTO_INCREMENT PRIMARY KEY

插入数据 INSERT INTO

语句：INSERT INTO sites (column1,column2) VALUES (value1,value2)
向sites表中插入一条记录



成功插入，并且id是自增的

批量插入 executemany()

使用executemany()方法批量插入数据，该方法第一个参数为sql语句，第二个参数为元祖列表，包含了我们要插入的数据（格式化的数据）
mydb.commit() 提交事物，事物在程序是默认开启的，当有数据表被修改时，必须执行这句代码
mycursor.rowcount 返回受影响的行数
例子：因为要插入的是多组数据，所以元祖里要放入列表来包含每组插入的数据


如果想要在这个记录修改后获取记录ID

使用mycursor.lastrowid
print('受到改变的记录ID值为：',mycursor.lastrowid)


可以看到插入后对应的ID值为12

查询数据，获取所有结果集 fetchall()

使用SELECT语句
myresult = mycursor.fetchall() 获取所有结果集

从结果集中只读取一条数据 fetchone()

两种方法：
1、myresult = mycursor.fetchone() 从结果集中读取一条数据的函数 fetchone()
2、sql语句加入LIMIT 1


使用通配符 like % 模糊查询

在数据库中模糊查询一个值，使用%
例子：从数据库中查找url地址中间是pp的记录

防范SQL注入攻击 %s

为了防止SQL注入攻击对数据库造成危险，在查询参数的时候使用占位符%s 来防范sql注入
%s
例如：
sql = 'SELECT * FROM sites WHERE name = %s'
value = ('baicai', )

排序 Order by x

order by name 默认升序，从小到大
order by id desc 降序

LIMIT 3 OFFSET x 指定从开始位置x+1往后读取3条记录

sql = 'SELECT * FROM sites LIMIT 3 OFFSET 1' # offset 1 意思是从第二条开始读取，limit 3 意思是读取三条
从查询结果的第二条开始读取往后读取三条

删除记录 DELETE FROM

sql = "DELETE FROM sites WHERE name='sbbb'"
这里影响了两行，表示有两条记录被删除掉了

为了防止SQL注入，在删除数据的时候也要转义传入的内容

更新表数据 UPDATE .. SET

更新语句也要防范SQL注入

删除表 DROP TABLE IF EXISTS tablename
