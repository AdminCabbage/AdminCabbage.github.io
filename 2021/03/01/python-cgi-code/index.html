CGI简介

CGI：通用网关接口，是一段程序，运行在HTTP服务器上，CGI程序可以是 Python脚本，PERL脚本，SHELL脚本，C脚本等语言构成。

服务器配置CGI

首先在apache的httpd.conf配置文件中设置网站cgi-bin目录
ScriptAlias /cgi-bin/ "D:/AppServ/www/cgi-bin/"
添加py后缀
AddHandler cgi-script .cgi .py
去掉注释
LoadModule rewrite_module modules/mod_rewrite.so
启动apache访问
127.0.0.1:8080/cgi-bin/hello.py
这就是我们写的cgi程序地址
